# 測試指南 (Testing Guide)

本文件說明如何執行自動化測試，以驗證「中古機報價系統」的功能是否正常。

## 測試範圍

測試腳本 (`tests/test_full_system.py`) 涵蓋了以下功能：

1.  **登入/登出**: 驗證密碼保護與 Session 管理。
2.  **管理後台**: 驗證是否能正確取得並進入隱藏的管理後台路徑。
3.  **報價單管理**:
    *   建立報價單 (`/generate_quote`)
    *   檢查報價單是否存在 (`/check_quote_exists`)
    *   單筆價格更新 (`/update_single_price`)
    *   刪除報價單 (`/delete_quote`)
4.  **頁面顯示**:
    *   驗證主頁面 **不顯示** PDF/LINE 按鈕。
    *   驗證靜態報價單頁面 **顯示** PDF/LINE 按鈕。
    *   驗證生成的 HTML 檔案內容正確。

## 如何執行測試

請在專案根目錄 (即 `app.py` 所在的資料夾) 開啟終端機 (Terminal / CMD / PowerShell)，然後執行以下指令：

```bash
python -m unittest tests/test_full_system.py
```

## 預期結果

如果一切正常，您應該會看到類似以下的輸出：

```text
........
----------------------------------------------------------------------
Ran 8 tests in 0.123s

OK
```

*   `OK` 表示所有測試都通過了。
*   如果有任何 `FAIL` 或 `ERROR`，請檢查錯誤訊息以進行除錯。

## 注意事項

*   測試過程會使用暫存資料夾，**不會** 影響您現有的 `static_quotes` 或 `mappings.json` 資料。
*   測試會模擬爬蟲資料，不會真的去爬取網站，所以執行速度很快且不需要網路連線。
